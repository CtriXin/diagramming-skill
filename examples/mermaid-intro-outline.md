# Mermaid Intro

## One-line summary

`diagramming-skill` defaults to Mermaid and generates durable local outputs.

## Structure

- Mermaid-first
- Outputs
- Excalidraw fallback

## Key relations

- Mermaid -> text-based diagrams
- Mermaid -> repo-friendly workflow
- Excalidraw -> optional hand-drawn presentation layer

## Notes

- Default output is `mmd + outline + html`
